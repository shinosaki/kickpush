# KickPush

[Kick Webhooks](https://docs.kick.com/events/introduction)のイベントをWebPushとして通知します。

## Motivation

KickのWebアプリは、WebPushを用いた通知に対応していません。

KickPushは、WebPush APIを用いてストリーマーごとのイベントを購読し、Notification APIを通じてユーザに通知を送信します。
